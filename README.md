## Installation  
To install the library for development.

### Synchronize the project dependencies
*Commands must be executed from the project root directory.*
    
   ```bash
   $ uv sync
   ```

#### Tests

   ```bash
   $ make test
   ```

#### Format

   ```bash
   $ make format
   ```

#### lint

   ```bash
   $ make lint
   ```
